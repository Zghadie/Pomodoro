# ⏳ Pomodoro Timer (Python + Tkinter)

A graphical Pomodoro timer built with Python’s Tkinter GUI library.

## 📌 Overview

This productivity timer uses the Pomodoro technique: 25-minute work sessions followed by 5-minute breaks. Built as part of Day 28 of the *100 Days of Code: Python Bootcamp*.

## 🛠️ Tech Used

- Python 3
- Tkinter

## 📚 What I Learned

- Creating GUI windows and widgets
- Using `after()` for timing
- Styling and layout with Tkinter
